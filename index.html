Privacy Policy — AureliaVelis

Effective date: July 16, 2026

This policy covers Etsy Shop Pin Publisher ("the App"), the personal pin-publishing tool of the Etsy shop AureliaVelis (etsy.com/shop/AureliaVelis). The App is a personal, owner-operated tool. It is used exclusively by its owner to publish pins of the owner's own Etsy shop products to the owner's own Pinterest Business account. The App is not offered to, and cannot be used by, any third party.

What data the App accesses

  • Etsy: publicly available listing data of the owner's own Etsy shop (product titles, descriptions, images, and listing URLs), retrieved via the official Etsy Open API.
  • Pinterest: the owner's own boards and pins, accessed via the official Pinterest API with the scopes boards:read, boards:write, pins:read, pins:write, solely to list boards and create pins on the owner's account.

How data is used and stored

  • Data is used for one purpose only: creating pins of the owner's Etsy products on the owner's Pinterest boards.
  • All processing happens locally on the owner's personal computer. A small local file records which products have already been pinned, to avoid duplicates.
  • API credentials and access tokens are stored locally on the owner's computer and are never transmitted to anyone other than Etsy and Pinterest themselves.

What the App does not do

  • It does not collect, store, sell, or share any personal data of any third party.
  • It does not access any Pinterest or Etsy account other than the owner's own accounts.
  • It does not use cookies, analytics, tracking, or advertising services.
  • It does not scrape websites and does not automate engagement (likes, follows, comments).

Data retention and deletion

The only stored data is the local pin history file and API tokens on the owner's computer. These can be deleted at any time by the owner, and Pinterest access can be revoked at any time from Pinterest account settings (Settings → Security → Connected apps).
